# Canvas Feature Roadmap

## Missing common Miro/Eraser-style features

- Rotate objects
- Group / ungroup
- Bring forward / send backward layering
- Object alignment tools
- Distribute spacing
- Alignment guides / smart guides
- Drag-to-marquee multi-select
- Lock / unlock objects
- Comments
- Sticky notes as a distinct tool/type
- Frames / sections
- Templates
- Export image/PDF/SVG
- Real-time collaboration
- Presence cursors
- Sharing permissions
- Version/history beyond local undo/redo
- Minimap
- Search/find objects
- Presentation mode
- More diagram-specific tools:
  - sequence diagrams
  - ERDs
  - cloud architecture icons
  - code/markdown blocks
- Shape rotation/advanced transforms
- Layer ordering UI
- Better paste from external text/images
- Context menu / right-click menu
- Object toolbar positioning near selected item
- Connector labels editing UX improvements
- Auto-layout / tidy-up diagram
- More connector routing options and handles
- Swimlanes / containers
- Shape libraries / icon libraries
- Image upload/drop
- File import/export beyond current JSON state

## Current focus

1. Groups
2. Layers / z-order
3. Alignment
4. Distribution
